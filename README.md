### 🔬 Current Work — TinyISPNet

Designing a **5.8K-parameter neural ISP** for the Sony IMX135 camera sensor that jointly performs RAW image denoising and HDR tone mapping, targeting 30+ FPS on mobile GPUs. The pipeline takes raw Bayer measurements straight to perceptually optimized RGB output, trained end-to-end with an L1 + SSIM perceptual objective across 2,120+ RAW/JPEG image pairs.
